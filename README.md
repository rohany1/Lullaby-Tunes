<h1 align="center">  Ninja Tunes Music Player</h1>
<h3 align="center"><i> A frontend design for a music player website using HTML and CSS</i></h3>

#### Languages Used:
1. HTML5
2. CSS
   - CSS variables
   - Flex Layout
   - Media queries for responsive design

- [x] Ninja Tunes Homepage
